# PPT 模板说明

## 模板体系

内置 8 套模板（2 产品线 × 4 阶段），按产品线和评审阶段自动选择。

### 产品线适用范围

| 前缀 | 适用范围 |
|------|---------|
| **hm** | 空调、公司通用平台技术 |
| **ml** | 美菱冰箱、洗衣机、小电日电、美菱数字化项目 |

### 模板清单

| 产品线 | 阶段 | 模板文件 |
|--------|------|---------|
| hm | Charter | `hm-charter-template.pptx` |
| hm | PDCP | `hm-pdcp-template.pptx` |
| hm | ADCP | `hm-adcp-template.pptx` |
| hm | 转移 | `hm-transfer-template.pptx` |
| ml | Charter | `ml-charter-template.pptx` |
| ml | PDCP | `ml-pdcp-template.pptx` |
| ml | ADCP | `ml-adcp-template.pptx` |
| ml | 转移 | `ml-transfer-template.pptx` |

## 使用规则

1. 模板根据产品线 + 阶段自动选择，无需用户指定
2. 所有公司级汇报均基于对应产品线的统一模板
3. 表格页只需填入数据行，表头和格式由模板提供
4. 结论"恳请IPMT批准..."为固定写法，不要修改
5. 每个阶段模板中的示例数据仅供参考格式，实际生成时会被替换
