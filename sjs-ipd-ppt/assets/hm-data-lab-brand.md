# 虹美数据技术所品牌规范

> 基于公司 PPT 模板提取的品牌预设。

## 配色

| 角色 | 色值 | 用途 |
|------|------|------|
| primary | #0070C0 | 主色，标题装饰、关键图标 |
| primary-dark | #0161B3 | 内容页背景色 |
| accent | #00A3E0 | 强调色，数据高亮 |
| accent-warm | #FF8C00 | 暖色强调，亮点数字、KPI |
| success | #10B981 | 正向指标 |
| warning | #EF4444 | 风险标记 |
| bg | #FFFFFF | 白色背景 |
| surface | #F0F7FA | 卡片/区块背景 |
| body-text | #1A1A2E | 正文文字 |
| muted-text | #6B7280 | 次要文字 |
| border | #E5E7EB | 边框、分割线 |

## 背景

| 页面类型 | 背景处理 | 文件 |
|---------|---------|------|
| 封面页 | 背景图片 | `assets/ppt起始背景.png` |
| 内容页 | 背景图片 | `assets/ppt内容背景.png` |
| 表格页 | 背景图片 | `assets/ppt内容背景.png` |

## 字体

| 角色 | 字体 | 字重 |
|------|------|------|
| title | Microsoft YaHei | Bold |
| body | Microsoft YaHei | Regular |
| emphasis | Microsoft YaHei | Bold |
| data | Arial | Bold |

## 页面布局

| 页面 | 布局 |
|------|------|
| 封面 | 居中标题 + 公司名 + 项目信息 |
| 目录 | 网格/列表展示章节 |
| 内容页 | 章节标题（居中）+ 子标题 + 正文/图表 |
| 表格页 | 章节标题 + 表格 |
| 结尾 | 致谢 |

## 语气

结论先行、数据驱动、专业严谨、简洁有力
