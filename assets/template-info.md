# PPT 模板说明

## 美菱PPT统一模板

**路径**: `~/SynologyDrive/虹美公司/部门/美菱PPT模板.pptx`
**更新**: 2026-06-09

### 技术规格

| 属性 | 值 |
|------|-----|
| 尺寸 | 16:9 宽屏（960×540pt） |
| 标题字体 | 等线 Light |
| 正文字体 | 等线 |
| 主色调 | 蓝色 #4472C4 |
| 主题 | Office 默认 |

### 11种可用布局

| 序号 | 布局名称 | 用途 | 建议使用场景 |
|------|---------|------|-------------|
| 0 | 标题幻灯片 | 封面/章节封面 | 封面、结尾 |
| 1 | 标题和内容 | 通用内容页 | 大部分内容页 |
| 2 | 节标题 | 章节分隔页 | 大章节切换 |
| 3 | 两栏内容 | 左右对比/图文 | 架构图+文字 |
| 4 | 比较 | 双栏对比 | 方案对比、计划vs实际 |
| 5 | 仅标题 | 纯标题页 | - |
| 6 | 空白 | 自由排版 | 自定义图表 |
| 7 | 内容与标题 | 左图右文 | 流程图+说明 |
| 8 | 图片与标题 | 图片展示 | 截图、效果图 |
| 9 | 标题和竖排文字 | 中文竖排 | 特殊排版 |
| 10 | 竖排标题与文本 | 竖排标题 | 特殊排版 |

### 使用规则

1. 所有公司级汇报均应基于此模板
2. 项目管理模板（CHARTER/PDCP/ADCP/转移汇报PPT）为基于此模板的细分场景版本
3. 制作汇报材料时优先使用此模板
